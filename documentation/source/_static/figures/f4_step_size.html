fanroots.step_size — Optimisers
Input
Proposed step direction p
current residual $r(h)=\|F(h)\|_2^2$  ·  gradient $\nabla r$  ·  learning rate η and momentum m
Step-size optimiser
naive
α = 1 always
shrink
halve α until
$r(h+\alpha p) \leq r(h)$
backtracking (BLS)
Armijo condition:
$r(h+\alpha p) \leq r(h)+c\alpha\nabla r\cdot p$
reduce α by β = 0.8
ternary
ternary search on α ∈ [0,1]
assumes unimodal residual
Output
Scale factor α ∈ [0, 1]
actual step: Δh = α·m·η·p
